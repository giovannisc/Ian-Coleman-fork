# Proposal: Bitcoin Best Practices Tutorial

## Purpose
Train users in operational routines that reduce real-world loss events.

## Target Audience
- Self-custody users preparing to hold real funds.
- Operators who need repeatable, auditable safety practices.

## Learning Outcomes
- Apply secure setup practices for devices and environments.
- Apply backup hygiene and recovery verification discipline.
- Reduce common mistakes involving exposure, storage, and social engineering.

## Suggested Module Outline
1. Secure environment setup (offline, clean machine, no third-party scripts).
2. Backup strategy and physical storage planning.
3. Recovery drills and periodic verification routines.
4. Incident response basics (suspected compromise, migration procedures).

## Suggested In-App Activities
- Operational checklist before deriving real wallets.
- Backup planning worksheet (without storing secrets in the app).
- Recovery drill reminders and logging template.
- Risk self-assessment questionnaire.

## Acceptance Criteria
- Users complete a pre-use safety checklist.
- Users can describe a recovery drill process end-to-end.
- Users can identify high-risk behaviors and safer alternatives.

## Open Questions for Discussion
- Required frequency for recovery drill prompts.
- Include optional enterprise/team operational variants.
- Provide printable checklists from the application or external docs only.
